## 📌 Descripción

**TechGame27 + CALCA** es una tienda online en desarrollo que combina:

- 🧥 Indumentaria deportiva (CALCA)
- 📱 Tecnología (TechGame27)
- 🛒 Carrito de compras funcional
- 🎞 Slider interactivo
- 📱 Diseño responsive

---

## ⚙️ Tecnologías utilizadas

- HTML5
- CSS3 (animaciones + responsive)
- JavaScript (vanilla)
- LocalStorage (persistencia de carrito)

---

## 🧠 Funcionalidades

✔ Agregar productos al carrito  
✔ Guardado automático en el navegador  
✔ Eliminar productos del carrito  
✔ Vaciar carrito completo  
✔ Cálculo de total  
✔ Slider automático con botones y dots  
✔ Menú responsive mobile  
✔ Animaciones UI modernas  
✔ Scroll suave
